# Legacy Scripts

Эти скрипты сохранены для обратной совместимости.

**ВАЖНО**: Используйте новый unified скрипт `atlastunnel.sh` для всех новых установок.

Старые скрипты больше не поддерживаются и могут быть удалены в будущих версиях.

## Старые скрипты

- `atlastunnel_pptp.sh` - PPTP VPN
- `atlastunnel_l2tp.sh` - L2TP VPN
- `atlastunnel_l2tp-ipsec.sh` - L2TP/IPsec VPN
- `atlastunnel_ikev2.sh` - IKEv2 VPN (без шифрования)
- `atlastunnel_ikev2-ipsec.sh` - IKEv2/IPsec VPN
- `atlastunnel_sstp.sh` - SSTP VPN

## Миграция на новую версию

Запустите `atlastunnel.sh` в корне проекта для установки unified решения.
